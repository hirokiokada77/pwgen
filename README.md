## Usage

`pwgen [OPTIONS]`

### Options

- `-l`, `--length <LENGTH>`: Password length [default: 20]
- `-p`, `--pattern <PATTERN>`: Character pattern for the password
  [default: A-Za-z0-9]
- `-n`, `--no-newline`: Do not print the trailing newline character
- `-h`, `--help`: Print help
- `-V`, `--version`: Print version
